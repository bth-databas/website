Här installerar vi databasen MariaDB på en Linux-maskin och i Windows WSL.



## Förberedelse

Du har en distribution av Linux installerat och uppdaterat. Guiden förutsätter att det är Ubuntu eller Debian men andra varianter fungerar (troligen) också.

Du är bekant med terminalen.

Principen är att installera med pakethanteraren så har du en annan Linux-distribution kan du behöva anpassa installationsstegen till den miljön du använder.



## MariaDB Server

För att få bakgrunden till MariaDB så läser man på i [dokumentationen för MariaDB](https://mariadb.org/documentation/).

MariaDB Server innehåller databasservern och terminalbaserade klientprogram.



## Installera MariaDB Server

Börja med att uppdatera pakethanterarens lista över paket.

```bash
sudo apt update
```

Om du inte redan har gjort det, kan du också uppgradera de paket som finns installerade. Det skadar sällan att göra det.

```bash
sudo apt upgrade -y
```

Installationen av databasservern sker med pakethanteraren, kör följande kommando i terminalen.

```bash
sudo apt install mariadb-server
```

Databasservern startar automatiskt som en daemon. Du kan dubbelkolla dess status, och stoppa och starta den, via kommandot `service`.

```bash
sudo service mariadb status
```

Nu är du redo att koppla upp dig mot databasservern.



## Koppla upp mot databasservern

För att koppla upp mot databasservern använder vi klientprogrammet `mariadb`. Just nu vill vi enbart verifiera att vi kan koppla upp oss så vi använder `sudo` för att koppla upp oss som root-användaren.

```bash
sudo mariadb
```

Om allt fungerar så ska du nu vara inloggad i MariaDB:s terminalgränssnitt. Du ser en prompt som ser ut ungefär så här:

```sql
MariaDB [(none)]>
``` 

Du kan skriva kommandot `SHOW DATABASES;` för att se vilka databaser som finns i databasservern. 


```sql
SHOW DATABASES;
```

För att lämna terminalgränssnittet för MariaDB så kör du kommandot `exit`.

```sql
exit
```



## Avslutningsvis

Du har installerat MariaDB Server på en Linux-maskin (eller Windows WSL) och verifierat att du kan koppla upp dig mot databasservern med klientprogrammet `mariadb`.
