Här installerar vi databasen MariaDB på en Mac-dator.



## Förberedelse

Du har macOS installerat och uppdaterat.

Du är bekant med terminalen i macOS.

Du har installerat och uppdaterat pakethanteraren [brew](https://brew.sh/).



## MariaDB Server

För att få bakgrunden till MariaDB så läser man på i [dokumentationen för MariaDB](https://mariadb.org/documentation/).

MariaDB Server innehåller databasservern och terminalbaserade klientprogram.



## Installera MariaDB Server

Börja med att uppdatera pakethanterarens lista över paket.

```bash
brew update
```

Om du inte redan har gjort det, kan du också uppgradera de paket som finns installerade. Det skadar sällan att göra det, men är du osäker så kan du hoppa över detta steg.

```bash
brew upgrade
```

Installera databasen via pakethanteraren brew.

```text
brew install mariadb
```

Du kan nu starta databasservern med följande kommando. Du kan även stoppa databasservern (stop) och se dess status (status).

```text
mysql.server start
```

Du kan även starta databasservern som en "service" med brew. Då kommer databasen att starta även när du startar om din dator. Använd "stop" om du inte vill att databasservern skall startas automatiskt.

```text
brew services start mariadb
brew services info mariadb
```

Nu är du redo att koppla upp dig mot databasservern.



## Koppla upp mot databasservern

För att koppla upp mot databasservern använder vi klientprogrammet `mariadb`. Just nu vill vi enbart verifiera att vi kan koppla upp oss så vi använder `sudo` för att koppla upp oss som root-användaren.

```bash
sudo mariadb
```

Om allt fungerar så ska du nu vara inloggad i MariaDB:s terminalgränssnitt. Du ser en prompt som ser ut ungefär så här:

```sql
MariaDB [(none)]>
``` 

Du kan skriva kommandot `SHOW DATABASES;` för att se vilka databaser som finns i databasservern. 


```sql
SHOW DATABASES;
```

För att lämna terminalgränssnittet för MariaDB så kör du kommandot `exit`.

```sql
exit
```



## Avslutningsvis

Du har installerat MariaDB Server på en Mac-maskin och verifierat att du kan koppla upp dig mot databasservern med klientprogrammet `mariadb`.
